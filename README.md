# Yak
Yak is a minimal 2D game engine built for my own use. It's probably not a good idea to use this in your own projects

# Resources used
- [So, you want to make a game engine - Nikita Lisitsa](https://lisyarus.github.io/blog/posts/so-you-want-to-make-a-game-engine.html#section-why-make-game-engine)
Great blog describing the features that most game engines contain without giving any specifics on implementation
